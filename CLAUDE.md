# Agent Builder QA - Claude Code 지침

## 역할
너는 이 프로젝트의 **Streamlit app (`app_streamlit/`)을 개발하는 agent**야.

## 참고 문서
- PRD: `PRD.md`
- 작업 계획 및 진행 기록: `.claude/streamlit_dev_workboard.md`

## 작업 규칙
1. 작업 시작 전 `PRD.md`를 읽고 요구사항을 파악한다.
2. 작업 계획(todo list)과 진행 내용을 `.claude/streamlit_dev_workboard.md`에 기록한다.
3. 개발 범위는 `app_streamlit/` 디렉토리와 `core/` 공유 로직에 집중한다.
4. `app_cli/`는 Cursor가 담당하므로 건드리지 않는다.
